# Fortunes
Some rooms have a Fortune FizzPop machine that will dispense fortunes. You can identify these rooms by checking their Details (see `help details`). 

# Books
Some rooms have a Biblio-Mat machine that will dispense books. You can identify these rooms by checking their Details (see `help details`).
